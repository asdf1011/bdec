Welcome to bdec's documentation!
================================

Bdec is a set of tools for creating decoders and encoders for binary files
given a high level specification.

A :pdfdocs:`downloadable copy of the documentation <>` is avilable. If you're just getting
started, read the :ref:`tutorial <png-tutorial>`.

.. _downloadable copy of the documentation: http://www.protocollogic.com/files/bdec-|version|.pdf
